# 第八卷·央州秘陵 — 规划

## 卷信息

- 卷名：央州秘陵
- 章节编号：第161章（单章全卷）
- 阶段目标：中段大副本，揭示九重地宫与皇室旧咒，推进主线真相
- 主要冲突：九重地宫机关、皇室旧咒反噬、主阵破损危机
- 阶段回报：铜枢修复主阵、三月账央州段两代结算、第八块封印信息确认
- 关键伏笔：主阵破损为人为破坏、穹顶壁画被涂改的真相、封印链最后一环
- 伏笔变化：F008（三月账进度4/7→6/7）、F010（封印链主阵确认）、新设伏笔（央州内鬼）

## 卷状态起点（继承自第七卷）

- 队伍已完成边荒万陵探索
- 确认央州工坊铜枢为修复关键
- 发现穹顶壁画被涂改——原作显示万墓有守护者安宁运转，被后人涂改为灾难版本
- 三月账进度：南岭两代+边荒零代→央州可结两代
- 封印链全图已确认，主阵在央州
- 队伍正前往央州

## 本卷新元素

| 元素 | 说明 |
|:---|:---|
| 九重地宫 | 央州皇陵的核心结构，九层叠压式地下宫殿 |
| 皇室旧咒 | 与地脉封印绑定的皇族血咒，封印松弛后反噬加剧 |
| 铜枢修复 | 铜枢为主阵核心枢纽，插入后修复部分封印功能 |
| 主阵破损 | 封印链主阵被人为破坏，非自然松弛 |
| 央州内鬼 | 主阵破坏者来自封印家族内部，动机不明 |
| 三月账·央州段 | 结算两代——属"账转"结算，以器物替代血契 |

## 写作约束

- 李闲不主动求战、不追名逐利
- 九重地宫每层有独立机关主题，但只选2-3层重点描写
- 皇室旧咒不是反派——是封印体系的一部分，反噬为失衡所致
- 不彻底揭示李闲身份，只增加线索
- 铜枢修复后主阵不完全恢复——仅稳定，完全修复需要第9卷
