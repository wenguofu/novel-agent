# 第九卷·地脉祖墓 — 规划

## 卷信息

- 卷名：地脉祖墓
- 章节编号：第162章（单章全卷）
- 阶段目标：终极揭秘——封印链原点、万墓根源、李闲来历关键线索
- 主要冲突：原始阴煞、祖墓中枢、内鬼真相、最后一代三月账
- 阶段回报：封印链完整修复、三月账全部结清、旧掌印来源确认、内鬼身份揭示、李闲身份线索
- 关键伏笔收官：F008（三月账6/7→7/7）、F010（封印链完整）、F024（内鬼）、F025（涂改真相）、F026（旧掌印）、F027（铸造者）
- 后置伏笔：第10卷人间闲游余韵

## 卷状态起点（继承自第八卷）

- 铜枢已归位，主阵稳定
- 三月账进度6/7，剩最后一代（地脉祖墓）
- 封印链被内部改写的真相已确认
- 铜枢上发现旧掌印（女子手印）
- 缺口深处刻有"内"字
- 封印谱系被人改动——中间缺一人
- 队伍离开央州，前往地脉祖墓方向

## 本卷新元素

| 元素 | 说明 |
|:---|:---|
| 地脉祖墓 | 封印链的物理原点——第一代封印者的葬地 |
| 原始阴煞 | 万墓异动的根源——与地脉封印同步诞生的自然力量 |
| 祖墓中枢 | 十代封印链的中央控制节点 |
| 初代封印者 | 第一个建立封印体系的人——遗留完整记录 |
| 真正的第七代 | 被从谱系中抹去的那个人——内鬼的根源 |
| 旧掌印主人 | 铜枢上的女子手印——与初代封印者的关系 |
| 李闲杯痕 | 第三次器物印记确认——三件器物上的同一印记指向同一答案 |

## 写作约束

- 不彻底揭示李闲全部来历——留白到第10卷做最终收束
- 不写李闲主动求战、追名逐利
- 原始阴煞不是反派——是自然力量
- 封印链完全修复，但恢复后地脉需要时间自然平复
- 内鬼身份揭示但不写因果的过度纠缠——真相本身够重
- 三月账全部结清，账册关闭
