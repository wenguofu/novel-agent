# 第三卷章纲：南岭蛊墓

## 卷级预排

- 第三卷预计章节数：80
- 覆盖章节：156-235
- 本卷总目标：从水源地出发沿山路进入南岭苗寨，接触蛊术体系，了解蛊母和旧仪，以蛊术替代路径完成三月账南岭段结算，获得第三块地脉封印信息，确立第四卷方向。
- 已加载规则文件：`genre_bible.md`、`world_bible.md`
- 本段主要限制：不提前揭示蛊母真身，不让配角完全掌握蛊术体系，不将蛊术与水道强行关联，不进黑水沉门，不让李闲主动求战。

## 第三卷起点状态（继承自第二卷）

- 队伍已启程前往南岭，预计三天路程
- 纸片方位图确定第一条路线
- 蛊母信息已知（存在超百年，与村寨有契约，非传统蛊术）
- 陈父约数十年前曾到访南岭收集信息
- 苏灵考据方法在南岭不适用
- 三月账进度 2/7，南岭可结两代

## 写作约束

- 正文写作 Agent 必须读取本文件中对应章节条目。
- 章节标题、核心事件、节奏规则不得擅自改变。
- 未登记到本文件的章节不得写作。
- 若剧情需要变化，由总主编剧 Agent 修改本文件。
- 每章正文须遵循 `genre_bible.md` 的单章模板：配角发现异常 → 专业判断显示高危 → 危机扩大 → 李闲出手 → 危机消散 → 配角认知变化 → 尾部新谜团。
- 蛊术相关描写不得完全对标水道体系——这是两套底层逻辑不同的文明。
- 蛊母不作为反派处理——作为中性守护力量。
- 配角登场必须服务于剧情推进，不得为填充篇幅而加配角。
