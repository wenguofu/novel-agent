# 第082章 危机详细

> 章节名称：旧柜外角再看

**危机类型**：设定确认·名签未脱

**危机触发**：陈远山临行前独自回到旧柜外角前。柜角仍闭，黑色水面静止如镜。

**侵蚀过程**：他靠近柜角时，手腕上的旧痕不跳不痛，安静得像一块褪色的印记。三月账的痕迹已经完全消失。

**当事人反应**：陈远山伸手触碰柜角表面，木头的触感干冷而坚实，不再有虚化感。

**临界状态**：名签仍贴在柜角上，背面那枚借名旧印的红点在晨光中反射出一丝极细的光。

**主角化解**：李闲没有过来，只在远处说了一句"放着又不会丢"。

**后果明确**：若不确认名签状态，水道中可能无法识别与旧柜相关的标记。


**化解后果**：旧柜外角维持关闭状态，名签未脱落。

**下章任务**：任务：三月账在水下有望结算——水道中的旧柜式标记可能激活水下分支。**原因**：需要确认危机是否完全解除，避免遗留隐患。**后果扩大**：若忽视，危机可能在下一次封印点探查时复发，且强度将超过本次。
