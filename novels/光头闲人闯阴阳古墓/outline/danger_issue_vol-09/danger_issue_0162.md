# 危机详细：第162章——地脉祖墓

## 危机一：原始阴煞暴走

**触发条件**：队伍进入核心墓室，扰动平衡
**危机升级过程**：
1. 核心墓室中央有一片透明区域——无色无形的气流在缓慢旋转
2. 苏灵的记录仪在进入区域后全部失灵
3. 空气温度从常温骤降至零下——三秒内地面结霜
4. 无色阴煞开始向队伍方向蔓延——视觉不可见，但能通过温度变化感知
5. 王硕左臂接触阴煞边缘——皮肤表面出现冻伤样的白色斑块
6. 陈远山用铜钱试探——铜钱在阴煞中直接被冻裂
7. 阴煞蔓延加速——四人脚下的地面全部结霜

**危险可见异变**：无色无形态——通过温度骤降和结霜感知；地面霜层呈现放射状扩散
**队伍成员被牵入边缘的过程**：
- 王硕左臂冻伤——剧痛但皮肤无伤口
- 苏灵的相机镜头结冰
- 陈远山铜钱碎裂——铜钱碎成片状，比被重物砸碎的更脆断

**陷入危险者的无力感**：
- 所有物理工具无效
- 铜钱无效——陈远山意识到这比任何阴物都古老
- 温度持续下降，无法阻止

**主角化解动作**：
- 李闲取下搪瓷杯
- 蹲下，将杯底压在地面霜层的中心——不是能量最强的位置，是扩散的圆心
- 杯底接触霜层的瞬间，霜层停止扩散——从中心开始融化
- 无色阴煞从暴走状态回归缓慢旋转

**化解后效果**：
- 温度回升至正常
- 霜层消融后地面无痕迹
- 阴煞恢复稳定循环

## 危机二：三月账最后一代——旧印反噬

**触发条件**：李闲将搪瓷杯放入石台凹槽
**危机升级过程**：
1. 石台感应到杯底后开始震动
2. 石台表面浮现暗金色纹路——从中心向四周扩散
3. 七代封印者的名字依次亮起——第一代至第六代的名称快速通过
4. 第七代位置亮起时——纹路从暗金色变成暗红色——旧印反噬
5. 暗红色纹路顺着石台蔓延到地面——温度升高
6. 石台周围的岩石开始发烫

**队伍成员被牵入边缘的过程**：
- 三人退到石台边缘——地面温度已达烫手程度
- 陈远山的鞋底在高温地面边缘发出焦味

**主角化解动作**：
- 李闲的手没有离开杯子
- 他按住杯底，向下轻压
- 杯底与石台接触处发出一声清脆的声响——像杯底嵌入了凹槽的最深处
- 暗红色纹路从边缘开始褪去——从暗红变回暗金
- 第七代名字的位置——暗金色纹路形成了一个完整的闭合圆环

**化解后效果**：
- 温度恢复正常
- 石台表面纹路全部转为暗金色
- 三月账七代全部结算完成
- 搪瓷杯杯底多了一圈金色纹路
