### 背景与身世

- 孤儿院长大：付大强是弃婴，在江城郊区的晨光福利院长大至18岁，自此独立生活。
- 性格成因：缺乏家庭关爱使他过早独立，不擅长社交但观察力极强，内心渴望联结却不善于表达。
- 学业选择：选择历史系是出于对「过去发生的事情」的执着——某种程度上，他想知道自己的来历。
- 经济状况：毕业后积蓄不多，暂住城中村民宿，靠打零工维持生活。
- **被系统选中的原因**：孤儿院被系统选中作为「容器培育点」，付大强从小就被系统以各种方式引导，塑造出适合承载神格的性格特质（独立、坚韧、善于观察）。他的「秩序」异能也是系统在胎儿期就埋下的种子。

| **反叛精神** | **面对系统的操控，会主动质疑并反抗（越到后期越明显）** |

## 女主

### 叶微

| 字段 | 内容 |
|:---|:---|
| 姓名 | 叶微 |
| 性别 | 女 |
| 年龄 | 外观约22岁（实际存活时间远超表面） |
| 身份 | 生命古神艾维娅的唯一容器宿主 |
| 异能 | 轮回——以生命印记为媒介重置时间线/记忆片段 |

### 背景与身世

- 表面身份：叶微看起来是乐园Lv.6层级的一名普通参与者，外表年轻，性格温和冷静，拥有出众的判断力和直觉。
- 真实身份：叶微是生命古神艾维娅的容器宿主，已死亡并重生过至少十次以上（具体次数是核心悬念之一）。
- 重生原因：艾维娅赋予了她轮回种子能力。当叶微死亡且羁绊足够强时，轮回种子会激活，将她的意识回溯到过去的某个节点——但每次重生都会消耗记忆作为代价。
- 重生次数越多，失去的记忆越多。她已经忘记了最初进入乐园的原因、自己的家人、甚至自己的名字是怎么来的。唯一没有被磨灭的，是她脑海中不断回响的一个模糊的身影——那是她在某一次重生中深爱的人，而那个人早已在时间线上消失。

### 核心特质

| 特质 | 描述 |
|:---|:---|
| 温柔而坚定 | 表面温和，内心有钢铁般的意志，不会为了自己而动摇 |
| 牺牲倾向 | 经历过太多次失去，对自己的生命价值评估极低，愿意为他人生存付出一切 |
| 洞察力极强 | 多次轮回积累了大量经验，能一眼看穿游戏规则和人心 |
| 刻意隐藏 | 刻意隐藏自己的真实能力，避免引起古神和主角的过度关注 |
| 深情隐忍 | 对付大强的感情来源于某一次轮回中建立的最深羁绊，但她选择隐忍不发 |

### 重生能力详解

轮回异能（艾维娅赋予）：

| 能力 | 描述 |
|:---|:---|
| 时间回溯 | 死亡后以生命印记为媒介，将意识回溯到过去的特定节点 |
| 记忆代价 | 每次重生消耗大量记忆，最终可能失去自我 |
| 信息优势 | 重生后保留前世大部分记忆，包括游戏规则、敌人弱点、危机预警 |
| 生命印记 | 艾维娅赐予的媒介，数量有限（叶微目前剩余约3-5枚，具体数量是悬念） |
| 羁绊触发 | 重生必须在羁绊足够强的情况下才能激活 |

### 与付大强的关系演变

| 阶段 | 关系 | 描述 |
|:---|:---|:---|
| 第1卷初遇 | 相遇 | 付大强在第一卷末尾首次见到叶微，她表现得像一个普通参与者 |
| 第2卷 | 暗中守护 | 叶微开始频繁出现在付大强身边，以顺路为借口帮他规避危机 |
| 第3卷 | 揭示真相 | 付大强意外发现叶微的重生能力，两人关系进入新阶段 |
| 第4卷 | 并肩作战 | 两人开始正式合作，叶微多次以重生记忆化解生死危机 |
| 第5卷 | 感情升温 | 付大强逐渐发现叶微的情感牵绊，开始追问她的过去 |
| 第6卷 | 献祭 | 叶微接受艾维娅的夺舍，以自己的存在换取付大强的生机 |
| 第7卷 | 灵魂对抗+永恒陪伴 | 叶微以灵魂碎片帮助付大强对抗叛神，最终以记忆形式永久存在于付大强意识中 |

### 终极命运：接受夺舍 + 神力转移

这是全文最核心的悲剧高潮。

叶微与艾维娅的关系：
- 艾维娅对叶微有双重期望：既希望她作为容器不断进化，又希望最终夺舍她的身体。
- 叶微与艾维娅之间存在一种独特的共生关系——艾维娅欣赏叶微的意志，因此给了她轮回能力和一定的自主权。
- 艾维娅的夺舍并非一次性完成，而是一个渐进过程。每当叶微使用轮回能力，她与艾维娅的融合度就会上升。

关键抉择（第6卷高潮）：当付大强面临足以让他彻底消亡的危机时，叶微面临两个选择：继续使用轮回躲避危机（但她已接近生命印记耗尽）or 主动接受艾维娅的夺舍，利用古神力量帮助付大强渡过难关。

叶微的选择：她选择了第二条路。

后手（第6卷中后期）：在被艾维娅完全夺舍之前，叶微利用与艾维娅建立的羁绊联结，将自己体内积累的大部分艾维娅神力剥离并转移给付大强。

结果：
- 付大强获得了艾维娅的神格碎片，权限和能力大幅提升
- 叶微的意识被艾维娅压制，但并未完全消亡——她的记忆作为灵魂碎片留存在付大强的意识深处

付大强成神（第7卷）：在终章，付大强融合了秩序异能 + 玄清传承 + 艾维娅神格碎片 + 其他古神的力量碎片，正式成神。**在对抗叛神夺舍的关键时刻，叶微的灵魂碎片觉醒，以记忆为武器与叛神意识对抗，成为付大强最终消灭叛神的关键力量。**
