# 合规名称规则

## 适用范围

所有项目资料、章纲、正文、审稿记录都必须遵守本规则。

## 禁止内容

不得出现真实地区、国家、省份、城市、领导人、名人名称。不得用谐音、缩写、拼音、外文原名绕过规则。

## 替代规则

- 现实世界母星统一称为：蓝星。
- 现实国家使用虚构国名，例如：夏国、鹰国、樱国、雪原联邦。
- 现实首都或主要城市使用虚构城名，例如：上京、首市、海州、南陵。
- 现实省份使用虚构区域名，例如：东洲、西岭、南域、北境。
- 现实领导人、政客、名人使用虚构称谓，例如：特不靠谱、白宫商人、红毯女王。
- 真实组织、真实公司、真实平台使用虚构名称。

## 别名表要求

所有替代关系必须登记到 `alias_registry.md`，正文只允许使用别名。

登记格式：

```text
类别：
别名：
说明：
使用范围：
首次出现章节：
```

## 审查规则

合规审查 Agent 必须执行以下检查：

1. 检查是否出现真实地名、真实国家名、真实省份名、真实城市名。
2. 检查是否出现现实领导人、政客、名人名称。
3. 检查是否出现真实公司、真实组织、真实平台名称。
4. 检查是否出现明显指向现实对象的谐音、缩写、外文名。
5. 将未登记替代名写入 `alias_registry.md`。

## 处理结果

合规结论只允许三类：

```text
通过：未发现违规名称。
修改：存在少量违规名称，已列出替代方案。
重写：现实指向过强，需要改写设定或剧情。
```
